¿Cómo funciona la app?

Dos calculadoras. Una para calcular la hipotenusa cuando el usuario da el valor de dos lados de un triángulo. La otra para calcular el valor de un lado, cuando el usuario da el valor de la hipotenusa y el otro lado. En el html puse dos forms que reciben los datos del usuario: ya sean los dos lados, o un lado y la hipotenusa. La información del form la recibe una función de Javascript que calcula el valor faltante con el teorema de Pitágotas. Además, se pusieron restricciones para la calculadora. Los lados no pueden ser negativos o valer 0, y tienen que ser números (no pueden ser letras, símbolos, etc).
